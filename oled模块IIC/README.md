# CH32V307 OLED 模块示例仓库

这个仓库整理了一个基于 `CH32V307VCT6` 的 `SH1106 128x64` 4 针脚 I2C OLED 示例，适合直接用 MounRiver Studio 打开、编译和下载。

## 目录说明

- `4针脚I2C版本/`
  - 最初参考的 4 针脚 OLED 代码来源。
- `第二讲/第二讲/`
  - 当前整理完成、适合继续开发和上传 GitHub 的主工程。

## 当前主工程特性

- 已切换为 `CH32V307` 平台
- 使用 `I2C1` 重映射到 `PB8/PB9` 的硬件 I2C 驱动 `SH1106`
- OLED 驱动、演示逻辑、主循环入口已经拆分模块
- 源码补充了中文注释
- 自带串口启动信息和多种 OLED 动画演示

## 默认接线

- `VCC -> 3.3V`
- `GND -> GND`
- `SCL -> PB8`
- `SDA -> PB9`
- `PA9(USART1_TX) -> USB 转串口 RX`
- `GND -> USB 转串口 GND`

串口波特率：`115200`
