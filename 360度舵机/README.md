# PWM 舵机 + EC11 控制说明

本工程已从循迹 ADC 示例改为 CH32V307 的 PWM 舵机驱动示例，并接入 EC11 旋转编码器控制 360 度连续舵机。

## 硬件连接

| 模块/信号 | 接法 |
| --- | --- |
| 舵机信号线 | PA8 / TIM1_CH1 |
| 舵机 VCC | 舵机外部 5V 供电 |
| 舵机 GND | 舵机电源 GND 与开发板 GND 共地 |
| EC11 A 相 | PC1 |
| EC11 B 相 | PC2 |
| EC11 公共端/GND | GND |

> 不建议直接用开发板 5V 给大扭矩舵机供电，舵机启动电流较大时容易让单片机复位。
> PC1/PC2 已在程序里配置为内部上拉输入。如果你的 EC11 模块板自带 VCC，请按模块丝印供电；裸 EC11 编码器通常 A/B 接输入，公共端接 GND 即可。

## PWM 参数

| 参数 | 当前值 |
| --- | --- |
| 定时器 | TIM1 |
| 输出通道 | CH1 |
| 输出引脚 | PA8 |
| PWM 周期 | 20ms / 50Hz |
| 停止脉宽 | 1500us |
| 360 舵机驱动范围 | 1000us ~ 2000us |
| 360 舵机速度分辨率 | 0.1ms / 100us |
| 普通角度舵机范围 | 500us ~ 2500us |
| 当前演示 | EC11 转动时舵机跟随转一小段，拧得越快舵机档位越高，停手后停止 |

## 文件结构

| 文件 | 作用 |
| --- | --- |
| `User/main.c` | 主程序，初始化串口、舵机 PWM、EC11，并轮询编码器控制舵机 |
| `HardWare/inc/servo_pwm.h` | 舵机 PWM 配置和 API 声明 |
| `HardWare/src/servo_pwm.c` | TIM1_CH1 PWM 初始化、脉宽、360 舵机档位控制实现 |

## 常用接口

### `Servo_PWM_Init()`

初始化 PA8 为复用推挽输出，配置 TIM1 为 1MHz 计数、20ms 周期 PWM。

### `Servo_SetPulseUs(uint16_t pulse_us)`

直接设置 PWM 高电平脉宽，单位是 `us`，范围会限制在 `500us ~ 2500us`。

### `Servo_Clockwise(uint8_t level)`

顺时针旋转，`level` 是 1~5 档：

| level | 脉宽 | 效果 |
| --- | --- |
| `1` | 1.4ms | 顺时针最慢 |
| `2` | 1.3ms | 顺时针稍快 |
| `3` | 1.2ms | 顺时针中速 |
| `4` | 1.1ms | 顺时针更快 |
| `5` | 1.0ms | 顺时针最快 |

### `Servo_CounterClockwise(uint8_t level)`

逆时针旋转，`level` 是 1~5 档：

| level | 脉宽 | 效果 |
| --- | --- | --- |
| `1` | 1.6ms | 逆时针最慢 |
| `2` | 1.7ms | 逆时针稍快 |
| `3` | 1.8ms | 逆时针中速 |
| `4` | 1.9ms | 逆时针更快 |
| `5` | 2.0ms | 逆时针最快 |

### `Servo_Stop()`

输出 1.5ms 脉宽，让 360 舵机停止。

### `Servo_SetSpeed(int8_t speed)`

兼容旧写法，会自动把 `-100~100` 转成 5 个档位。正数走逆时针，负数走顺时针，`0` 停止。

### `Servo_SetAngle(uint8_t angle)`

仅用于普通 180 度角度舵机，将 `0 ~ 180` 映射到 `500us ~ 2500us`。360 度连续舵机不能用它定位角度。

## 当前上电现象

程序启动后串口输出舵机 PWM 信息，然后：

1. `Servo_PWM_Init()` 初始化 `PA8 / TIM1_CH1`
2. `EC11_Init()` 初始化 `PC1 / PC2` 为上拉输入
3. `Servo_Stop()` 先输出 `1.5ms`，舵机保持停止
4. 主循环每 `1ms` 读取一次 EC11
5. 旋钮顺时针转一格，舵机顺时针转约 `80ms` 后停止
6. 旋钮逆时针转一格，舵机逆时针转约 `80ms` 后停止
7. 连续两格之间间隔越短，舵机速度档位越高

## EC11 速度档位

程序会根据两次有效旋转之间的时间自动选择 `1~5` 档：

| 两格间隔 | 舵机档位 |
| --- | --- |
| `<= 80ms` | 5 档，最快 |
| `<= 140ms` | 4 档 |
| `<= 240ms` | 3 档 |
| `<= 400ms` | 2 档 |
| `> 400ms` | 1 档，最慢 |

如果要调“多快算快”，修改 `User/main.c` 里的 `EC11_LEVEL5_INTERVAL_MS`、`EC11_LEVEL4_INTERVAL_MS`、`EC11_LEVEL3_INTERVAL_MS`、`EC11_LEVEL2_INTERVAL_MS`。

如果每转一格舵机动得太少或太多，修改 `User/main.c` 里的 `SERVO_RUN_AFTER_STEP_MS`。

如果旋钮方向和舵机方向相反，把 `User/main.c` 里的 `EC11_REVERSE_DIRECTION` 从 `0U` 改成 `1U`。
